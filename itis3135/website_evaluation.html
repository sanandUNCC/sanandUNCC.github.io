Website Evaluations

Bad Website: Yale School of Arts

Summary: This webpage was built for the Yale School of Arts program and its students.
It has many images and captions that display the multiple art projects made by students at the school.

  • C: The contrast is only in this website because there are opposing text colors against the background.
    The background colors for each blocked element made readability a bit diffucult for some headings.
  • R: This website has a varying color scheme such that there doesn't seem to be any continuity in font color and backgrounds.
  • A: The elements of this page are all aligned to the left side and they only flow to each other through the grey bar that goes from the top to bottom of the page.
  • P: All the elements in this website are relatively close together with almost no space to separate them.
    It shoul seem that all the containers relate to each other but that is not the case so keeping them all together is a bad move.

Good Website: SuperCoach Courier Mail

Summary: This webpage was built for the NRL SuperCoach fantasy rugby competition that is sponsored by KFC.
It allows for the visitor to get caught up on the latest news and even sign up to be a part of the fantasy league journey.

  • C: The contrast between the dark fonts and light backgrounds allows for better readability for the viewer.
    The shading behind the containers allows for them to be seen clearer against the background of the page.
  • R: This page has a sense of continuitiy as it follows a single font for most if not all of its elements.
    The color scheme for the whole webpage is white, green, black, and gold.
  • A: This page's content for the most part, is centered while the extras are aligned to the right.
    The elements flow easily as the center container has divisions that show different sections all related to the overarching topic.
  • P: This website has all related topics in one container such they are in close proximity to each other.
    This allows for there to be space between them and other containers that have other ideas.
